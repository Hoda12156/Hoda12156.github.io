Hardware




Hardware: la parte fisica del computer

La parola hardware deriva dall’inglese (“hard” significa pesante e “ware” significa oggetto) e significa letteralmente “ferramenta”.
L’hardware è tutto ciò che si può toccare in un computer.
È diverso dal software, che è la parte “invisibile” (i programmi).
L’hardware è il corpo del computer, fatto di schede, cavi e periferiche.
La scheda madre è il cuore del sistema: collega tutto e permette il funzionamento.
Senza hardware, il software non può fare nulla.
L’hardware comprende: schede, cavi, periferiche, processori, memorie… tutto ciò che serve per far funzionare il sistema.

Schede



Nel computer ci sono alcune schede elettroniche molto importanti.
Le schede servono per controllare e gestire le varie funzioni del computer: immagini, suoni, rete, memoria…
Sono tutte collegate alla scheda madre, che fa da base.

1) Scheda Madre (la più importante)
È la base di tutto il computer.
Tutti i componenti principali (CPU, RAM, scheda video, dischi, porte USB…) sono collegati direttamente alla scheda madre.
Ha dei circuiti elettronici che permettono ai vari componenti di parlarsi tra loro.

Contiene:
Socket: dove si inserisce la CPU (il “cervello” del computer).
Slot RAM: per le memorie.
Slot PCIe: per aggiungere schede (video, audio, rete…).
Chipset: gestisce le comunicazioni interne.
Porte: USB, HDMI, Ethernet, ecc.

Senza scheda madre, le altre parti del computer non possono comunicare, quindi non funziona nulla.

2) Scheda Video (o Scheda Grafica)
Gestisce tutto ciò che riguarda le immagini.
Mostra il desktop, i giochi, i video sul monitor.
Alcuni computer hanno una scheda video interna alla scheda madre.
I computer per giocare o lavorare con la grafica usano schede video dedicate (più potenti).
Le marche più famose sono NVIDIA e AMD.

3) Scheda Audio
Gestisce i suoni del computer.
Permette di sentire l’audio con le casse o le cuffie.
Permette anche di registrare suoni con un microfono.
Oggi è quasi sempre integrata nella scheda madre.

4) Scheda di Rete
Serve per collegarsi a internet o a una rete locale (LAN).
Può essere via cavo (Ethernet) o senza fili (Wi-Fi).
Anche questa scheda è spesso integrata nella scheda madre.

5) Schede di espansione
Sono schede aggiuntive che si possono inserire nella scheda madre per aggiungere funzioni.
Esempi:
Scheda TV (per vedere la TV sul PC)
Scheda di acquisizione video (per registrare da videocamere o console)
Scheda USB aggiuntiva (per avere più porte USB)

Cavi

I cavi servono per: Alimentare (cioè portare corrente elettrica) al computer.
Trasmettere dati tra le varie parti.
Tipi di cavi:
  • Cavo di alimentazione: collega il computer alla presa di corrente.
  • Cavi SATA: collegano hard disk e SSD alla scheda madre.
  • Cavi USB: servono per collegare mouse, tastiere, stampanti, chiavette.
  • Cavi HDMI/VGA/DisplayPort: collegano il computer al monitor.
  • Cavo Ethernet: collega il PC al modem/router per andare su internet.

Periferiche

Le periferiche sono i dispositivi collegati al computer per comunicare con lui.
Le periferiche sono gli oggetti che si collegano al computer per inserire dati, ricevere dati, o fare entrambe le cose.

Periferiche di Input (inseriscono dati nel PC):
Tastiera → scriviamo testi, comandi.
Mouse → spostiamo il cursore, clicchiamo.
Microfono → inviamo suoni al computer.
Scanner → trasforma un foglio cartaceo in un file digitale.
Webcam → manda immagini/video al computer.

Periferiche di Output (ricevono dati dal PC):
Monitor → mostra ciò che succede nel computer.
Stampante → stampa documenti su carta.
Casse → emettono suoni e musica.

Periferiche di Input/Output (fanno entrambe le cose):
Touch screen → schermo che possiamo toccare (input) e vedere (output).
Chiavetta USB → riceve e invia dati al PC.
Stampante multifunzione → stampa (output), scansiona (input) e copia.
Hard disk esterni → possono essere usati per salvare o leggere file.

Software




Il software è l’insieme di istruzioni e dati che permettono a un computer o a un qualsiasi dispositivo elettronico di funzionare.
A differenza dell’hardware (la parte fisica, come tastiera, CPU, monitor), il software è la parte invisibile: i programmi, le applicazioni, i sistemi operativi.
Il software dice all’hardware cosa fare.
Senza software, un computer è solo un pezzo di metallo, plastica e circuiti.
Il software viene creato da sviluppatori (o programmatori), che scrivono codice usando linguaggi di programmazione come:
  • Python
  • C++
  • Java
  • JavaScript
Questi linguaggi permettono di istruire il computer a svolgere compiti ben precisi.

Tipi principali di Software

Firmware



È un tipo di software di basso livello incorporato direttamente in un hardware.
Si trova in microchip o memorie flash all’interno dei dispositivi.
Controlla le funzionalità base dell'hardware e permette la comunicazione tra hardware e software.
Non viene quasi mai modificato dall’utente.

Esempi:
Il firmware del BIOS/UEFI nel PC: avvia il computer prima che il sistema operativo parta.
Il firmware di una stampante che gestisce il movimento delle testine.

Sistemi operativi

È il software fondamentale che fa da intermediario tra l’hardware e gli altri software.
Gestisce risorse come memoria, processore, file e dispositivi.
Fornisce un’interfaccia utente per controllare il computer (ad es. desktop, finestre, pulsanti).

Esempi comuni:
Windows (Microsoft)
macOS (Apple)
Linux (open source)
Android (Google)
iOS (Apple, per iPhone)

Funzioni principali del sistema operativo:
Avviare e spegnere il dispositivo.
Avviare le applicazioni.
Gestire la sicurezza e gli utenti.
Controllare le periferiche (stampanti, mouse, ecc.).

Applicazioni

Sono i programmi che permettono all’utente finale di fare qualcosa di specifico.
Aiutano l’utente a scrivere documenti, navigare, modificare foto.
Si basano sul sistema operativo per funzionare.

Esempi:
Microsoft Word (videoscrittura)
Google Chrome (navigazione web)
Adobe Photoshop (fotoritocco)
Spotify (ascolto musicale)
